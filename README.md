# PyQt5 Calculator (Simple Addition App)

## 📌 Overview

This is a simple GUI-based calculator built using PyQt5.
It performs basic addition of two numbers entered by the user and displays the result in a label.

---

## 🚀 Features

* User input using `QLineEdit`
* Button click event handling
* Addition of two numbers
* Output display using `QLabel`
* Input validation using try-except

---

## 🛠️ Technologies Used

* Python
* PyQt5

---

## 🧠 Learning Outcomes

* Understanding GUI development in PyQt5
* Working with widgets like QLineEdit, QPushButton, QLabel
* Handling events (signals and slots)
* Basic error handling in Python
